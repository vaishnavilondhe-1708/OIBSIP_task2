# Customer Segmentation Analysis (iFood Dataset)

## Overview
This project performs customer segmentation analysis for an e-commerce business using customer purchase behavior data. The goal is to group customers into meaningful segments to support targeted marketing and better business decisions.

## Dataset
- iFood customer dataset  
- Includes customer income, recency, and spending across multiple product categories

## Approach
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature scaling  
- Customer segmentation using K-Means clustering  
- Visualization of customer segments  

## Tools Used
- Python  
- Google Colab  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

## Outcome
Identified distinct customer segments such as high-value customers, inactive users, and price-sensitive buyers, enabling actionable business insights.
